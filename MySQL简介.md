# 什么是MySQL
  * MySQL是一种DBMS（数据管理系统）
  * DBMS可以分为两类：
    * 基于共享文件系统的DBMS，包括Microsoft Access和FileMaker。通常桌面用途，不用于高端或更关键的应用。
    * 基于客户机-服务器的DBMS，包括MySQL，Oracle以及Microft SQL Sever等。
      * 服务器部分负责所有数据访问和处理的一个软件。
      * 客户机是用于打交道的软件。
